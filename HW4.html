Problem 1: Finding \(\pi\) in Rcpp

In this exercise, we will revisit Problem 1 in HW3 to find \(\pi\), but now you are asked to implement functions to estimate \(\pi\) by Monte-Carlo simulations using Rcpp and potentially RcppArmadillo. Moreover, instead of the graphical display, in this problem we will focus on improving the execution time of this program by translating it in C++.

In HW3, you already programmed a function called find_pi(), where it generates B points based on which you estimate \(\hat{\pi}\). Now you are asked to use the same approach to estimate \(\hat{\pi}\), but instead, you should implement a function called find_pi_cpp() as well as other auxiliary functions in a separate .cpp file called find_pi_rcpp.cpp. You can then source this file in your HW4 R script using the following command:
